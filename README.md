### Bot Features
```md
[ Moderations ]
- Kick
- Ban
- Timeout

[ Stocks ]
- View current stock price
- Frequent Viewed Stocks

[ Infomations ]
- Bot Info
- Server Info
- User Info
- Load Avatar

[ Logging ]
- Message Edit / Delete
- Member Join / Exit
- Role Permission / Feature Change
- Channel Create / Delete / Update
- Role Give / Delete

[ etc ]
- Random Draw
- Giveaways
```

If you use this Discord Bot, **make a ".env" file in your folder.**

And write this => `DISCORD_TOKEN=Your_Bot_TOKEN`

You need to turn on this options in [ Discord Developer Portal ] .
+ Server Members Intent
+ Presence Intent
+ Message Content Intent


[Discord Developer Portal]: https://discord.com/developers/applications/943838140316155974/bot
